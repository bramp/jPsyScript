Command groups: the slider

The slider looks like this:

It allows the user to select a value from a continuous scale. If the slider is being shown while the script is waiting for something, the user can adjust it any number of times. The slider is always oriented horizontally. You can label positions along the slider by positioning an image immediately above or below it.

The slider starts off in an unknown position, with an unknown setting, with an unknown minumum and maximum, and hidden. Consequently it's a good idea to use the 'move slider' and 'set slider' commands before you use the 'show slider' command for the first time.

To read the current setting of the slider use the '$slider' variable. You can read the current setting of the slider even while the slider is hidden.

One way to use the slider is like this:

      -- set up the 'OK' button

      load an OKButtonImage.png into cell A

      move cell A to (180,-150)

      show cell A



      -- set up the slider

      move slider to (0,-150) with size (300,30)

      set slider to 50 from 0 to 100

      show slider



      -- wait for a response and log it

      wait for a click in A

      log $slider

      log $lastClickTime



      -- remove the clutter from the display

      hide cell A

      hide slider

Commands

move slider to (x-pos,y-pos) with size (x-size,y-size)
x-pos,y-pos -- position in pixels,of middle of the slider relative to the middle of the window
x-size,y-size -- size of the slider in pixels
Positions the slider in the window. When a script starts running, the slider could be positioned anywhere so it's a good idea to use this command before you first use 'show slider'.
Examples:move slider to (0,-150) with size (300,30)

set slider to value [from minimum to maximum]
value -- the starting (default) position of the thumb
minimum -- the value corresponding to the left end of the scale
maximum -- the value corresponding to the right end of the scale
Changes the current setting of the slider. This command comes in two forms -- with and without the 'minimum' and 'maximum' parameters. When a script starts running, the scale could represent any range of values so you're recommended to use the longer form before the first time you need to show the slider. If you use the other form later in your script, your previous settings for 'minimum' and 'maximum' will remain unchanged.
Examples:set slider to $varP from 1 to 7
set slider to 50

show slider
Makes the slider visible. Does not change its position or current setting.

If you don't use 'move slider' before using this command the slider may be in a strange position, even somewhere offscreen where you can't see it.

If you don't use the longer form of 'set slider' before using this command the results you get from reading the slider position will be on an unpredictable scale.

hide slider
Makes the slider invisible. Does not change its position or current setting.
