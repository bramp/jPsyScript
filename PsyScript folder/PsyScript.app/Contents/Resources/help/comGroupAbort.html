Command groups: abort

The abort command causes execution of the script to stop as if the last command in the script had finished. PsyScript immediately jumps to the stage of presenting the script's log and allowing it to be saved to a file. This command is used mostly for debugging but it can be useful in scripts which do a small amount of testing before deciding whether the subject is suitable for the main part of the experiment.

Commands

abort
Makes PsyScript finish execution immediately rather than execute the rest of the script.
