Command groups: sound stimulus

This command plays a sound. The script doesn't continue until the sound has finished playing.

Commands

play sound theFile
theFile -- the file that the sound is stored in
Plays the sound.
Examples:play sound cheering.aiff

Technical notes

There may be a short pause (less than a second) after the sound has finished before the script realises it and carries on.

This command can currently play sounds in any of the formats suggested by these extensions:

acc, aiff, au, mp3, m4a, snd, wav

If you are creating a new sound it's probably best to use AIFF which is an uncompressed format. If you find that sounds in this format take up too much disk space you might like to try compressing them into acc, mp3 or m4a format.
