Command groups: the text field

The text field allows the subject to type in arbitrary text. If the text field is being shown while the script is waiting for something, the subject can type into it. You can include instructions by putting them into an image next to the text field.

The text field starts off in an unknown position, with unknown contents. Consequently it's a good idea to use the 'move text field' and 'set text field' commands before you use the 'show text field' command for the first time.

To read the current contents of the text field use the '$textField' variable. You can read the contents of the field even while it's hidden.

One way to use the text field is like this:

      -- set up the 'OK' button

      load OKButton.png into cell A

      move cell A to (180,-150)

      show cell A



      -- set up the text field

      move text field to (0,0) with size (300,30)

      set text field to $blank

      show text field



      -- wait for a response and log it

      wait for 60 seconds or a click in A

      log $textField

      log $lastClickTime



      -- remove the clutter from the display

      hide cell A

      hide text field

Commands

move text field to (x-pos,y-pos) with size (x-size,y-size)
x-pos,y-pos -- position of text field in pixels, relative to the middle of the window
x-size,y-size -- size of the text field in pixels
Positions the text field in the window. When a script starts running, the text field could be positioned anywhere so it's a good idea to use this command before you first use 'show text field'.
Examples:move text field to (0,-150) with size (300,30)

set text field to value
value -- the text to appear in the text field
Changes the text in the text field. When a script starts running, the text field could have any contents so use this before the first time you need to show the text field.
Examples:set text field to $varT
set text field to Type your answer here.

show text field
Makes the text field visible. Does not change its position or contents.

hide text field
Makes the text field invisible. Does not change its position or current setting.

Technical notes

For some reason you need to hit the tab key or click inside the text field before you can type in it. The program should be doing this automatically when the text field is shown but it's not happening. This is a bug.

You cannot change the font, font size, or colour of the text in the text box. This is not a bug.
