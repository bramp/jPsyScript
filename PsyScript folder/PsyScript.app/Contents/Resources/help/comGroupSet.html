Command groups: set

There are three forms of the set command:

        set <variable> to <value>

	

	set <variable> by appending <value>

	

	set <variable> by removing <value>

In each case, the variable to be set is one of the twenty six variables indicated by $varA to $varZ. It can be set to any of the following:

  • a fixed piece of text
  • the current value of another of those twenty six variables, or
  • any of these values as detailed in logging values:

    $dateStamp, $timeStamp, $space, $blank
    $lastClick, $lastClickTime, $lastKey, $lastKeyTime, $lastTime, $timer, $countdown, $slider, $textField

Commands

set variable to value
Replaces the existing value of the variable with a new one.
Examples:set $varL to $slider
set $varA to Hello,
set $varB to $lastClick

set variable by appending value
Appends the text indicated by value to the end of the text currently in the variable.
Examples:set $varC by appending $lastClick
set $varA by appending $varN

set variable by removing value
Removes all occurences of the value from the contents of the variable.
Examples:set $varL by removing $lastKey
set $varA by removing polite

Technical notes

The by removing form removes all occurences of the value, not just the first one: set QUEUEING by removing UE would result in 'QING'.

To clear a variable (i.e. to make it blank) you can use set $varV to $blank.

The variables you set are the same as the ones set by the 'repeat using' command.

You can't set the values of $timer, $countdown, or $slider using this command. Use their individual setting commands instead. See also

  • tables and using them
