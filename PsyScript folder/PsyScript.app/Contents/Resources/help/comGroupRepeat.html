Command groups: repeat

There are three forms of the repeat command:

        repeat while <condition>

            ...

	end repeat

	

	repeat until <condition>

	    ...

	end repeat

	

	repeat using <variables> from [<number> rows of] <table>

	    ...

	end repeat

The first two work roughly the same as each other and are discussed on this page. The last form is a way of accessing data from tables and is discussed in the section on tables and using them.

The two forms discussed here both take a condition as a parameter. You can read about what you can use as the condition in the section on conditions.

Commands

repeat while condition
    ....
end repeat
The commands between the repeat while line and the end repeat line are executed repeatedly until the condition becomes false.
Examples:repeat while $varL is in ABC
    ....
end repeat

repeat until condition
    ....
end repeat
The commands between the repeat until line and the end repeat line are executed repeatedly until the condition becomes true.
Examples:wait for a click in ABC
repeat until $lastClick is C
    ....
    wait for a click in ABC
end repeat

Technical notes

repeat blocks can be nested to any number of levels (until you run out of memory).

The test happens at the top of the block, not the bottom: if the condition is the wrong way around when the repeat command is encountered, the commands between those two lines are not executed even once.
