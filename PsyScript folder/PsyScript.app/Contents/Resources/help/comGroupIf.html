Command groups: if

The simplest form of the if command is

        if <condition>

            ...

	end if

In that form, the commands inside the block are executed only when the condition is true. You can read about what you can use as the condition in the section on conditions. The next simplest form is

        if <condition 1>

            ...

        else if <condition 2>

            ...

	end if

You can have as many conditions as you like:

        if <condition 1>

            ...

        else if <condition 2>

            ...

        else if <condition 3>

            ...

        else if <condition 4>

            ...

	end if

and only one of the blocks of commands will be executed: whichever one follows the first true condition. The last else of any set doesn't need to have an if attached to it:

        if <condition 1>

            ...

        else if <condition 2>

            ...

        else if <condition 3>

            ...

        else

            ...

	end if

In this case, the commands in the last block will be executed if none of the conditions is true: using an else line means that whatever the conditions exactly one set of commands in the if block will be executed.

Commands

if condition
    ....
[[else if condition
    ....]]
[else
    ....]
end if
The commands between the if line and the end if line are executed if the condition is true.
Examples:if $varL is A
if $varL is in BCDE
if c is in $varK

Technical notes

if blocks can be nested to any number of levels (until you run out of memory).
